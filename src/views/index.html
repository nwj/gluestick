Gluestick is a self-hosted pastebin that makes it easy to share code, notes, and text snippets.

Get Gluestick on

Grow Your Own Digital Garden

By compiling to a single executable and storing all its data in a single file, Gluestick is designed for easy self-hosting. And it runs at over 1,500 requests per second with sub-200ms P95 latencies on some of the cheapest, bottom-of-the-barrel cloud servers in the world.

Put Your Code's Best Foot Forward

Beautiful syntax highlighting support for hundreds of programming languages and text formats, via Syntect and the lovely Catppuccin Frappe colorscheme.

Outrageously Scriptable

Gluestick ships with a full-featured JSON API so you can programmatically create, read, and manage pastes with ease.
