Simple Self-Hosting

Gluestick compiles to a single executable, embeds its own assets and dependencies, and stores all its data in a single file for easy backup. And it runs at over 1500 requests per second with sub-200ms P95 latencies on some of the cheapest, bottom-of-the-barrel cloud servers in the world.

Put Your Code's Best Foot Forward

Beautiful syntax highlighting support for hundreds of programming languages and text formats, via Syntect and the lovely Catppuccin colorscheme.

Outrageously Scriptable

Gluestick ships with a full-featured JSON API so you can programmatically create, read, and manage pastes with ease.
